Multiplayer Blackjack add-on for Stake app

About

Multiplayer Blackjack is an add-on game for the Stake application. This game can not be installed on its own without having Stake installed first.

System requirements

  • Latest version of Stake

Installation

Step 1

Log in to the Stake backend.

Step 2

Open Add-ons page, find the purchased add-on in the list and click Install button.

Step 3

Input the purchase code and click Proceed button.

The add-on should be automatically downloaded and installed. Refresh the page in a browser, so the app loads and initializes the new add-on.
